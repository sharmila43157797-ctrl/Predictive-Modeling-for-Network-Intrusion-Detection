# Predictive Modeling for Network Intrusion Detection

## Overview

This project focuses on detecting malicious network traffic using Machine Learning techniques on the CICIDS2017 dataset. The model is trained to classify network intrusions based on network traffic features and evaluate its performance using standard machine learning metrics.

## Dataset

* CICIDS2017 (Cleaned Dataset)
* 52 Network Traffic Features
* Multi-class Network Intrusion Detection

> Note: The dataset is not included in this repository due to its file size. Please add the dataset file (`cicids2017_cleaned.csv`) to the project directory before running the notebook.

The developed machine learning model successfully classifies normal and malicious network traffic and can be applied in predictive network intrusion detection systems.
